About this project

Animals is an independent archive of speculative organisms — a long-term project exploring perception, form, and the space between images.

It has no commercial backing. Everything here is made by one person, slowly, over time.

How support helps

Your support helps keep this archive open, maintained, and growing.

It allows more time for observation, more works to be recorded, and more concepts to be developed — without rushing.

GitHub Sponsors

If this project means something to you — if something moved in your mind while watching — consider supporting through GitHub Sponsors.

There is no pressure. No reward tiers. Just a quiet way to say: continue.
